    Guía de Instalación#

    Esta guía te ayudará a instalar y configurar MkDocs en tu sistema para comenzar a crear documentación técnica profesional.

    Requisitos previos#

    Antes de instalar MkDocs, asegúrate de tener:

    • Python 3.8 o superior
    • pip (gestor de paquetes de Python)

    Verificar instalación#

    python --version
pip --version

    Instalación de MkDocs#

    Instalación básica#

    pip install mkdocs

    Instalación con tema Material#

    Para obtener la mejor experiencia visual, instala también el tema Material for MkDocs:

    pip install mkdocs-material

    Plugins recomendados#

    pip install mkdocs-git-revision-date-localized-plugin
pip install pymdown-extensions

    Crear un nuevo proyecto#

    Una vez instalado MkDocs, puedes crear un nuevo proyecto:

    mkdocs new mi-proyecto
cd mi-proyecto

    Esto creará la siguiente estructura:

    mi-proyecto/
├── mkdocs.yml
└── docs/
    └── index.md

    Configuración inicial#

    Edita el archivo mkdocs.yml para personalizar tu proyecto:

    site_name: Mi Documentación
site_description: Documentación de mi proyecto
site_author: Tu Nombre

theme:
  name: material
  language: es

nav:
  - Inicio: index.md
  - Guías: guias/
  - Referencia: referencia/

    Verificar la instalación#

    Para verificar que todo funciona correctamente:

    mkdocs serve

    Abre tu navegador en http://127.0.0.1:8000 para ver tu documentación.

    Puerto ocupado

    Si el puerto 8000 está ocupado, MkDocs automáticamente probará el siguiente puerto disponible.

    Solución de problemas comunes#

    Error: "command not found"#

    Si recibes este error al ejecutar mkdocs:

    1. Verifica que Python esté en tu PATH
    2. Reinstala MkDocs: pip install --upgrade mkdocs
    3. Usa el comando completo: python -m mkdocs serve

    Error de permisos#

    En sistemas Unix/Linux, si tienes problemas de permisos:

    pip install --user mkdocs

    Actualizar MkDocs#

    Para mantener MkDocs actualizado:

    pip install --upgrade mkdocs mkdocs-material

    Próximo paso#

    Una vez que tengas MkDocs instalado y funcionando, puedes continuar con la guía de uso básico.
