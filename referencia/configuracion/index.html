    Configuración Avanzada#

    Opciones de configuración avanzadas para personalizar tu sitio de documentación.

    Configuración del tema#

    Paleta de colores#

    theme:
  name: material
  palette:
    # Tema claro
    - media: "(prefers-color-scheme)"
      scheme: default
      primary: indigo
      accent: indigo
      toggle:
        icon: material/brightness-auto
        name: Cambiar tema
    # Tema oscuro
    - media: "(prefers-color-scheme: dark)"
      scheme: slate
      primary: indigo
      accent: indigo
      toggle:
        icon: material/brightness-7
        name: Cambiar tema

    Características del tema#

    theme:
  name: material
  features:
    # Navegación
    - navigation.tabs
    - navigation.sections
    - navigation.expand
    - navigation.top
    - navigation.footer
    - navigation.tracking

    # Búsqueda
    - search.suggest
    - search.highlight
    - search.share

    # Contenido
    - content.code.copy
    - content.code.annotate
    - content.code.select

    # Otros
    - content.tabs.link
    - content.action.edit
    - content.action.view

    Plugins avanzados#

    Git revision date#

    plugins:
  - git-revision-date-localized:
      fallback_to_build_date: false
      locale: es
      type: date
      enable_creation_date: true

    Minificación#

    plugins:
  - minify:
      minify_html: true
      minify_css: true
      minify_js: true

    Awesome pages#

    plugins:
  - awesome-pages:
      filename: .pages
      collapse_single_pages: true
      sort: true

    Extensiones de Markdown#

    Configuración completa#

    markdown_extensions:
  # Extensiones básicas
  - admonition
  - attr_list
  - def_list
  - footnotes
  - md_in_html
  - toc:
      permalink: "#"
      title: En esta página

  # PyMdown extensions
  - pymdownx.arithmatex:
      generic: true
  - pymdownx.betterem:
      smart_enable: all
  - pymdownx.caret
  - pymdownx.details
  - pymdownx.emoji:
      emoji_generator: !!python/name:material.extensions.emoji.to_svg
  - pymdownx.highlight:
      anchor_linenums: true
      line_spans: __span
      pygments_lang_class: true
  - pymdownx.inlinehilite
  - pymdownx.keys
  - pymdownx.magiclink
  - pymdownx.mark
  - pymdownx.smartsymbols
  - pymdownx.snippets:
      check_paths: true
  - pymdownx.superfences:
      custom_fences:
        - name: mermaid
          class: mermaid
          format: !!python/name:pymdownx.superfences.fence_code_format
  - pymdownx.tabbed:
      alternate_style: true
  - pymdownx.tasklist:
      custom_checkbox: true
  - pymdownx.tilde

    Configuración de búsqueda#

    plugins:
  - search:
      separator: '[\s\-,:!=\[\]()"/]+|\.(?!\d)|&[lg]t;|(?!\b)(?=[A-Z][a-z])'
      prebuild_index: true
      indexing: full

    Configuración de SEO#

    # Metadatos del sitio
site_name: Mi Documentación
site_description: Documentación técnica completa
site_author: Tu Nombre
site_url: https://tu-sitio.com

# Configuración de Google Analytics
google_analytics:
  - G-XXXXXXXXXX
  - auto

# Configuración de robots.txt
extra:
  version:
    provider: mike

    Configuración de despliegue#

    GitHub Pages#

    # Configuración para GitHub Pages
repo_url: https://github.com/usuario/repositorio
edit_uri: edit/main/docs/

    Netlify#

    # Configuración para Netlify
site_url: https://tu-sitio.netlify.app

    Configuración de desarrollo#

    # Configuración para desarrollo local
dev_addr: 127.0.0.1:8000
strict: true
use_directory_urls: true

    Variables de entorno#

    Puedes usar variables de entorno en tu configuración:

    site_name: ${SITE_NAME:-Mi Documentación}
site_url: ${SITE_URL:-https://localhost:8000}
